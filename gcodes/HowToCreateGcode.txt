# Inkscape

1. Create a document of required dimension
2. Add object

3. select object and path->object to path
4. select extensions->gcode tools -> tools library and select cone
	set diameter to 1
	feed to 200
	penetration feed 200
	depth 1
5. extensions -> gcode -> orientation to point
	select 2 point
	depth to -1
	unit to mm
6. extensions -> gcode -> path to gcode
	set path in preferences
